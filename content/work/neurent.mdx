# This is the main title.

Element Bolt is an AI-powered tool within the Element451 platform, designed to help higher education institutions automate and personalize communications with prospective students. While the platform utilizes AI to streamline workflows and automate tasks, my role as a UX Designer focused on designing new sections of the platform that aligned with the Bolt Design System and enhanced the user experience.

- Problem: Element Bolt needed a consistent, scalable design system to streamline workflows and improve engagement across multiple user touchpoints.
